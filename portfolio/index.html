Joystick Controlled Prosthetic Hand

A cardboard hand that contains Arduino to indivdiually move each of the hands fingers. The joystick is directly wired to a breadboard that is linked to two mini servos. Each of those servos controlls two to three fingers with strings to simulate muscle fibers. This project started as a way to explore the possibilties of Arduino while making something that I found interesting!

Technical Stack: C++, Arduino, and TinkerCAD

Process & Challenges

  • Created the cardboard hardware and used my prior knowledge to implement a minimal software.
  • Designed the project on TinkerCad to approach and eliminate possible errors.
  • Faced challenges related to how to move the fingers with low powered servos, overcame from researching similar projects.

def parse_line(line):
    try:
        time, level, msg = line.split(" | ", 2)
        return {"time": time, "level": level, "msg": msg}
    except:
        return None
