Joystick Controlled Prosthetic Hand

A cardboard hand that contains Arduino to individually move each of the hand's fingers. The joystick is directly wired to a breadboard that is linked to two mini servos. Each of those servos controls two to three fingers with strings to simulate muscle fibers. This project started as a way to explore the possibilties of Arduino while making something that I found interesting!

Technical Stack: C++, Arduino, and TinkerCAD

Process & Challenges

  • Created the cardboard hardware and used my prior knowledge to implement minimal software.
  • Designed the project in TinkerCad to identify and eliminate potential errors.
  • Faced challenges related to moving the fingers with low-powered servos, and overcame them by researching similar projects.

Code Snippet

#include 

Servo servo1;
Servo servo2;

void setup() {
  Serial.begin(9600);
  servo1.attach(7);
  servo2.attach(8);

  pinMode(6, INPUT_PULLUP);  
}

void loop() {
  int joystickValueX = analogRead(A0);
  int joystickValueY = analogRead(A1);
  int pushPin = digitalRead(6);

  int servoAngle = map(joystickValueX, 0, 1023, 0, 180);
  int servoAngle2 = map(joystickValueY, 0, 1023, 0, 180);
  servo1.write(servoAngle);
  servo2.write(servoAngle2);

  if (pushPin == LOW) {  // button pressed
    servo1.write(180);
    servo2.write(180);
    delay(1000);
  }
}

The goal was to develop a solution for the urban heat island effect in a chosen area, and we chose New York City. Code Snippet

import rasterio
import matplotlib.pyplot as plt
import numpy as np

heatmap_raster_path = "C:\\Users\\echoe\\OneDrive\\Desktop\\Geospatial Tech 25-26\\heat_map-main\\heat_map-main\\data\\output\\f_deviation_smooth.tif"
with rasterio.open(heatmap_raster_path) as src:
    heat_map = src.read(1)
    
    if src.nodata is not None:
        heat_map = np.ma.masked_equal(heat_map, src.nodata)
    else:
        heat_map = np.ma.masked_less(heat_map, -100)

    plt.figure(figsize=(9, 8))
    

    plt.imshow(heat_map, cmap='turbo')
    
    plt.title("NYC Urban Heat Map (Mean Surface Temperature 2020–2022)")
    
    plt.colorbar(label="Deviation from the Mean")
    plt.tight_layout()
    plt.grid()
    plt.show()
